HTML Basics

HTML stands for HyperText Markup Language.

It structures the web content using elements and tags.

<!DOCTYPE html> <html></html>
  • Headings: <h1> to <h6>
  • Paragraphs: <p>
  • Links: <a href>

HTML5 introduced semantic tags like <main>, <article>, <section>.

Tags can have attributes to define behavior or styling hooks.

CSS Styling

CSS stands for Cascading Style Sheets.

It controls the appearance of HTML elements.

body { font-family: Arial; } h1 { color: blue; }
  • Selectors
  • Properties
  • Values

CSS can be internal, external, or inline.

Use classes and IDs to target specific elements.

JavaScript Fundamentals

JavaScript adds interactivity to web pages.

It is a client-side scripting language.

let x = 10; function greet() { alert("Hello!"); }
  • Variables
  • Functions
  • Events

Modern JavaScript uses ES6+ features like arrow functions and let/const.

It integrates with HTML and CSS using the DOM.

DOM Manipulation

The DOM is a programming interface for web documents.

It represents the page so programs can change the structure, style, and content.

document.getElementById("demo").innerText = "Hi";
  • Access elements
  • Change content
  • React to events

Use querySelector for more flexibility.

Event listeners handle user interactions.

Responsive Design

Responsive Design ensures web content looks good on all devices.

Media queries adjust styles for different screen sizes.

@media (max-width: 600px) { ... }
  • Flexible grids
  • Flexible images
  • Media queries

Use relative units like %, em, and rem.

Mobile-first approach is preferred.
